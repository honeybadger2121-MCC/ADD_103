Crop

Goal: remove distractions and improve composition (rule of thirds, subject framing).

Resize

Goal: scale the image to an appropriate display width (e.g., 1200px hero, 800px content, 400px thumbnail).

Rotate

Goal: fix horizon tilt or adjust orientation.

Flip

Goal: create a mirrored composition or correct camera orientation if needed.

Exposure / Contrast

Goal: improve clarity with basic tonal adjustments (exposure, contrast, shadows/highlights).

Format / Compression

Goal: export to an efficient web format (JPG or WebP) and reduce file size while preserving quality.
