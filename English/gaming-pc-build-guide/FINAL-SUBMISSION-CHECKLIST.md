### If Images Don't Appear in PDF:

- Try different browsers (Chrome, Edge, Firefox)
- Use Microsoft Print to PDF instead of Save as PDF

### If Layout Looks Wrong:
- Adjust scale to 90-95% in print settings
- Check margins are set to Default or 0.5 inch
- Ensure paper size is Letter (8.5 x 11)

- If larger, you can reduce image quality or use PDF compression tools
