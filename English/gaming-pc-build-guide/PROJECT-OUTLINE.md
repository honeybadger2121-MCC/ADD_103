## Design Principles (CRAP)
- **Contrast:** Bold headings, color-coded steps, highlighted warnings
- **Repetition:** Consistent fonts, step numbering, icon usage
- **Alignment:** Clean margins, justified text, aligned images
- **Proximity:** Related information grouped, clear sections
