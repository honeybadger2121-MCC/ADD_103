2. Accordion

Accordions are a collection of collapsible items, perfect for FAQs, documentation, and organizing related content sections.

Bootstrap FAQ Accordion

Bootstrap is a powerful front-end framework that provides a collection of CSS and JavaScript components for building responsive websites and web applications. It includes a responsive grid system, pre-built components, and utilities for rapid development.

Bootstrap's JavaScript components use data attributes to initialize functionality without writing custom JavaScript. Components like modals, tooltips, carousels, and accordions can be activated using simple HTML attributes like data-bs-toggle and data-bs-target.

Bootstrap 5 introduced several major improvements including removal of jQuery dependency, new utility classes, improved grid system with CSS Grid support, new components like Offcanvas, updated color palette, and enhanced customization options with CSS custom properties.

Bootstrap can be customized through CSS custom properties (variables), Sass variables, utility classes, or by overriding default styles. You can also use online tools like Bootstrap Build or customize and download your own version from the official Bootstrap website.
