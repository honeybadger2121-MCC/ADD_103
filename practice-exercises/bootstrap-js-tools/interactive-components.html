Interactive Components

Explore Modal dialogs, Tooltips, and Popovers for enhanced user interaction and feedback.

1. Modal Dialogs

Modals are dialog boxes that overlay the main content, perfect for forms, confirmations, or detailed information.

Modal Examples
Basic Usage:
<button data-bs-toggle="modal" data-bs-target="#myModal">Open Modal</button>

2. Tooltips

Tooltips provide helpful hints and additional information when users hover over elements.

Tooltip Examples

Hover over these elements:

Badge Link with tooltip
Basic Usage:
<button data-bs-toggle="tooltip" title="Tooltip text">Hover me</button>

3. Popovers

Popovers are similar to tooltips but can contain more content including titles, body text, and even HTML.

Popover Examples
Special Popover Features:
Basic Usage:
<button data-bs-toggle="popover" data-bs-title="Title" data-bs-content="Content">Click me</button>

Modal Sizes

Bootstrap provides different modal sizes:

  • modal-sm - Small modal
  • Default - Regular modal
  • modal-lg - Large modal (this one)
  • modal-xl - Extra large modal
Best Practices
  • Use modals sparingly to avoid overwhelming users
  • Always provide a clear way to close the modal
  • Consider mobile responsiveness
  • Use appropriate sizes based on content
