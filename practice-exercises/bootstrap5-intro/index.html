This page demonstrates the core components of Bootstrap 5 framework

Container Example 1: Standard Container

This is a standard container that provides a responsive fixed-width container. It has max-widths at different breakpoints and centers content horizontally.

Container Example 2: Fluid Container

This is a fluid container that spans the full width of the viewport at all breakpoints. Notice how it extends to the edges of the browser window.

Container Example 3: Breakpoint-specific Container (container-lg)

This is a container-lg that behaves like container-fluid until the large (lg) breakpoint, then becomes a fixed-width container. Resize your browser to see the effect.

Container Example 4: Small Breakpoint Container (container-sm)

This is a container-sm that is fluid until the small (sm) breakpoint, providing more control over responsive behavior at different screen sizes.
