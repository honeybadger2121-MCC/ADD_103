Table Elements Explained

  • <table> - The container for the entire table
  • <thead> - Groups header content
  • <tbody> - Groups body content
  • <tr> - Table row
  • <th> - Table header cell
  • <td> - Table data cell
  • colspan - Spans multiple columns
  • rowspan - Spans multiple rows
