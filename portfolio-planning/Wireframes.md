## Common Elements Across All Pages
- **Header:** Site branding + horizontal navigation menu
- **Navigation:** Home | About | Projects | Skills | Contact
- **Footer:** Navigation links + copyright notice
- **Images:** Minimum 6 per page using figure/figcaption
- **Typography:** Clear hierarchy with H1, H2, H3 headings
- **Layout:** Mix of full-width content, floated images, and grid galleries
